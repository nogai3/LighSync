Режим разработчика (-dev_mode).

Хотите просмотреть переменные сцены, перемещения, FPS, а также техническую информацию?

Тогда вам дорога в режим разработчика. Откройте файл fnag1 по пути: Win+R -> %appdata% -> MMFApplications -> fnag1

И отредактируйте параметр -dev_mode=0 и замените 0 на 1. И всё! В меню вы увидете что DEVELOPER MODE активирован.

Заходите в ночь и радуйтесь!

**Примечание**: Режим-разработчика доступен только в версиях из веток: Snapshot, Pre-Release, Development/Test Build, в релизных версиях

Режим разработчика не будет доступен, думаю это логично.

Встречайте! Night-паки.

Это что-то типо дата-паков из Minecraft, только здесь вы устанавливаете с расширением .nightpack,

значения переменных игры (ночи), которые позволяет изменить API. Всего изменить можно 33 переменных.

Если вы хотите оставить значение переменной стандартным. Просто впишите значение из комментария после "default".

Как установить Night-пак?

Скачиваем файл имя (обязательно) ставим: FNAGCNP.nightpack | Далее: Win+R -> %appdata% -> MMFApplications. И перекидываем туда файл, затем

Заходим в игру и в главном меню нажимаем L. И автоматически грузится ночь.

Примечание: в 25w10d и ниже. Функция отключена.

Как создать Night-пак?

Скачиваем по ссылке nightpack_api.api, самой последней версии (ну если вы делаете всё для последней версии игры)

И редактируем на свой вкус и цвет. Установить и проверить его можно по способу выше.
